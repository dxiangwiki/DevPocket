# DevPocket
开发者口袋工具箱
纯原生 HTML + CSS + JavaScript
零框架 · 零后端 · 零依赖 · 纯本地浏览器运行

## 🧰 内置工具
1. JSON 格式化 / 压缩
2. 时间戳 ↔ 日期 双向转换
3. Base64 编码 / 解码

## ✨ 项目特点
- 轻量极简，代码干净易读
- Tab 标签页切换，交互简洁
- 所有运算本地完成，无需网络请求
- 支持静态部署，永久在线
